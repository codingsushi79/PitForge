rF2 Shared Memory Map Plugin

This DLL is distributed with PitForge Link under the plugin author's free
distribution terms. Place a copy of rFactor2SharedMemoryMapPlugin64.dll in
pitforge-link/resources/ before building the installer, or download from:
https://www.mediafire.com/file/s6ojcr9zrs6q9ls/rf2_sm_tools_3.7.15.1.zip/file
